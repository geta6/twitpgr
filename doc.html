About

Measure consciousness.
by @geta6

Usage

If hashtag or normal search then submit text with NO-MARK.
If measure the account tweet then submit text with @.
If measure the single text then submit text with $.
If account is locked, no results be found.

Point

point < __0 : low
point < 180 : normal
point < 500 : higher
point over 500 : madness

API(笑)

  • "q" : Will analyze a single text, json returned.
    • points : ISHIKI point.
    • length : text length.
    • kcodes : KANJI length.
    • ucodes : avarage of Unicode.
    • nwords : negative words.
    • pwords : positive words.
    • object : text object.
